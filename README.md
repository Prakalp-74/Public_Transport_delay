# Public Transport Delay Prediction

## About the Project
This project predicts public transport delays (in minutes) using machine learning.  
It uses factors like traffic, weather, and events to understand what affects delays.

---

## Goal
- Predict delay time  
- Identify important factors affecting delays  

---

## What I Did
- Cleaned and prepared the dataset  
- Converted categorical data using encoding  
- Built models:
  - Linear Regression  
  - Random Forest  
- Evaluated using MAE, MSE, and R² score  

---

## Results
- MAE: ~5 minutes  
- R²: ~0.5 (initially higher due to data leakage, later corrected)

---

## Challenges
- Faced data leakage issue  
- Model performance dropped after fixing data  
- Debugged using Git  

---

## Learnings
- Importance of clean data  
- Feature selection matters  
- Real-world data is not always perfect  

---
